import Highlight from '@site/src/components/Highlight';

- Type: <Highlight> `(name: string, url: string) => void | false | Promise<void | false> | Promise<CustomFetchReturnType>;` </Highlight>
  - 该 `hook` 的参数分别为：应用名、当前请求的资源地址；
  - `hook`返回值类型CustomFetchReturnType
    - code：请求url返回的代码字符串
    - type：请求url时响应头`Content-Type`内容
    - size：code字符串长度
  - **注意!!!**：只有当`hook`返回的内容为`Promise<CustomFetchReturnType>`，并且内容符合预期时，才会使用该`hook`返回的内容执行后续操作，否则loader会使用内部的request逻辑重新请求资源
- Kind: `async`, `sequential`
- Trigger:

  - garfish内部调用loader.load加载资源时触发

- 示例

```ts
Garfish.run({
  ...,
  loader: {
    fetch(name, url) {
      return void | false | Promise<void | false> | Promise<CustomFetchReturnType>;
    }
  }
})
```
